📋 How This Works

  • Everyone gets the same questions at the same time — answer fast and correctly for the most points.
  • Points fade the longer you take: answer within the grace period for full points, then the value decays down to a floor the slower you are.
  • Some questions are worth ⚡ 2x points — those are drawn from the toughest categories overall.
  • If "Read questions aloud" is on, everyone must wait for the same countdown before answering — that keeps it fair no matter how fast anyone reads.
  • Correct answers in a row build a 🔥 streak.
  • The battle ends when all questions are done, or earlier if someone hits the winning score.
  • You can leave anytime with Quit Battle; the host can end the battle for everyone with End Battle.
